# 📚 School Database (SQL Study Project)

This project was created for the purpose of practicing **relational database modeling** and **SQL concepts**, simulating the structure of an academic school system.

The focus of this project is to demonstrate knowledge of:

- Database design
- Table relationships
- Primary and Foreign Keys
- Data integrity rules
- SQL best practices

---

## 🏫 Scenario

The database represents a school that offers:

- Elementary School (1st to 9th grade)
- High School (1st to 3rd grade)

---

## 📏 Business Rules

The following rules were considered:

- Elementary School students can enroll only in **basic courses**
- High School students can enroll in **basic, technical, and vocational courses**

---

## 🧩 Database Structure

The system was modeled using the following core entities:

- **students**
- **courses**
- **classes**
- **enrollments**
- **classrooms**
- **assessments**

Relationships between entities were implemented using **FOREIGN KEY constraints** to ensure consistency.
